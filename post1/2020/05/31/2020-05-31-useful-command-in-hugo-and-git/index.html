Hugo与Git的常用命令

2020-05-31
Personal Notes

文章目录

    • Hugo常用命令
    • Git常用命令

Hugo常用命令

  1. hugo new site {siteName} 站点名称
  2. hugo new post/{name}.md 创建新的文章
  3. hugo server -D 以开发模式运行测试服务器
  4. hugo 编译网站
  5. hugo list drafts 列出草稿

Git常用命令

  1. git init 将当前文件夹初始化为git库
  2. git status 当前git库状态（查看修改、添加、删除了哪些文件）
  3. git add . 将所有文件加入Git库
  4. git commit -m “{Comment}” 提交版本
  5. git remote add origin {remoteGitAddress} 添加远程库地址
  6. git push -u origin master 将本地库与远程的Master分支绑定，并提交
  7. git push origin master 只提交到master分支，不绑定
  8. git pull –rebase origin master 将远端库与本地库同步（git push不上去时先试试这个）
  9. git push -u origin master -f 强制push（以本地库为准，覆盖掉远端库的所有Commit记录和代码）不可逆！请小心操作
  10. git branch {name} 建立新的分支
  11. git push -u origin {name}   push到指定分支
  12. git config –global user.name {name} 设置提交人用户名
  13. git config –global user.email {email} 设置提交人email
