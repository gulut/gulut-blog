GitHub Pages托管个人静态网站教程

本教程仅粗略地谈谈大概步骤，如需详细步骤，可以参考其他教程

步骤

1.创建一个GitHub账号 链接在此：https://github.com/join 注：GitHub的注册验证码如果一直无法通过的话，可以试试科学上网

2.创建一个Repositories 如果你是要放在{username}.github.io下。Repo 的名字就要为{yourusername}.github.io。否则访问时要加上Repo的名称。 就像：{username}.github.io/{repoName}/

3.用Git把网页上传到你的repo 此处教程十分多，就不写了

4.点开你的Repo选择Settings 在Options里找GitHub Pages，将Source从None改成master branch。

5.Done 如果你想要更高级的一些玩法，可以去看看Hexo和Hugo。

Tips：默认页面名称是index.html
